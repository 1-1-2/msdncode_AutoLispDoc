明经通道 AutoLISP 函数  
startapp
 

启动 Windows 应用程序

(startapp appcmd[file])

参数

appcmd

字符串，指定要执行的应用程序描述字串。如果 appcmd 没有包含全路径名，startapp 将按照环境变量 PATH 设置的路径来搜索该应用程序。

file

字符串，指定要打开的文件的名称。

返回值

如果成功则返回大于 0 的整数，否则返回 nil。

示例

下列代码启动 Windows 记事本，并打开 acad.lsp 文件。

命令： (startapp "notepad" "acad.lsp")

33

如果一个参数内含空格，它必须用双引号引起来。例如，用记事本编辑 my stuff.txt 文件需要使用下列语法：

命令： (startapp "notepad.exe" "\"my stuff.txt\"")

33
