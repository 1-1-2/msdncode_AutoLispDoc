明经通道 外部定义命令  
scene
 

（仅适用于图纸空间）创建新场景，修改或删除现有的场景（外部定义：render ARX 应用程序?/p> 

(c:scene mode [options])

参数

mode

字符串，有效值为?/p>

D 删除现有的场?/p>

L 列出图形中的所有场景，或返回指定场景的定义

M 修改现有的场?/p>

N 创建新场?/p>

R 重命名现有的场景

S 设置当前场景

options

有效值取决于指定?mode?/p>

D—删除场景

删除现有的场景?

(c:scene "D" name)

参数

name

字符串，指定要删除的场景的名称?/p>

如果删除的场景是当前场景，则??”将成为当前场景?/p>

示例

(c:scene "D" "PLANVIEW")

L—列出场景

列出图形中的所有场景，或返回指定场景的定义?/p> 

(c:scene "L" [name])

参数

name

字符串，指定要列出的场景的名称。如?name 参数被忽略，?c:scene 列出图形中定义的所有场景?/p>

返回?/p>

指定 name 时，c:scene 返回指定场景的定义?/p>

示例

如下代码列出图形中定义的场景名称?/p> 

命令?(c:scene "L")
("" "SCENE1" "SCENE2" "SCENE3")

空字符串 ("") 表示缺省场景??”，该场景不能进行修改?/p>

如下函数调用返回指定场景的定义：

命令?(c:scene "L" "SCENE2")
(T T) 
("VIEW1" nil)
("VIEW2" ("LIGHT1" "LIGHT2"))

M—修改场景

修改现有的场景?/p> 

(c:scene "M" name [view [lights]])

“修改”模式的选项与“新建”模式的选项相同。但在“修改”模式时，可以把参数 view 设置?nil，从而仅修改光源?/p>

注意 即使仅指定一个光源，也必须将 lights 参数设置为列表?/p>

例如，如下调用修改名?SCENE1 的场景，使之使用命名视图 FRONT 和图形中的所有光源：

(c:scene "M" "SCENE1" "FRONT" (C:LIGHT "L"))

如下调用修改场景 SCENE1，使之使用命名视?BACK 以及光源 P1 ?P2?/p> 

(c:scene "M" "SCENE1" "BACK" '("P1" "P2"))

N—新建场景

创建新场景?/p> 

(c:scene "N" name [view [lights]])

参数

name

字符串，指定新场景的名称?/p>

view

字符串，指定 AutoCAD 命名视图；或符号 T，指定?当前*”视图?/p>

lights

lights 参数的有效值为?/p>

  • 字符串列表，包含要使用的光源名。在只有一个光源的情况下也必须使用列表指定。在图形中的缺省值为 *全部* ?/li>
  • 符号 T，指定图形中的?全部*”光源?
  • Nil，指定不使用图形中的光源?/li>

示例

要创建新场景 DEFAULT，使之使用?当前*”视图和?全部*”光源，可使用如下函数调用：

(c:scene "N" "DEFAULT")

要创建新场景 DULL，使之使用?当前*”视图和缺省光源“在肩膀上”，可使用如下调用：

(c:scene "N" "DULL" T nil)

要创建新场景 SPECIAL，使之使用命名视?MY_VIEW 以及光源 SUN、LAMP ?SPOT，可使用如下函数调用?/p> 

(c:scene "N" "SPECIAL" "MY_VIEW" '("SUN" "LAMP "SPOT"))

R—重命名场?/p>

重命名现有的场景?/p> 

(c:scene "R" old_name new_name)

参数

old_name

字符串，指定原场景的名称?/p>

new_name

字符串，指定场景的新名称?/p>

示例

将场景的名称从“SPECIAL”改为“BRIGHT”：

(c:scene "R" "SPECIAL" "BRIGHT")

S—设置场景

设置当前场景?/p> 

(c:scene "S" [name])

参数

old_name

字符串，指定要置为当前的场景的名称?/p>

返回?/p>

如果忽略 name 参数，则 c:scene 返回当前选定场景的名称?/p>

示例

获取当前选定场景的名称：

命令?(c:scene "S")
"PLAN"

如果没有当前场景，则 c:scene 返回空字符串 ("")?/p>

要将 SCENE3 置为当前场景，可使用如下函数调用?/p> 

(c:scene "S" "SCENE3")
