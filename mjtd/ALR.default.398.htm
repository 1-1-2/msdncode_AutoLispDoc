vlr-lisp-reactor
 

构造编辑器反应器对象，通知 LISP 事件

(vlr-lisp-reactor data callbacks)

参数

data

任何要与反应器对象相关联的 AutoLISP 数据。如果没有数据则为 nil。

callbacks

表，其中包含下列格式的点对：

(event-name . callback_function)

在这里，event-name 是下表 “LISP 反应器事件”中的符号之一，callback_function 为符号，代表事件发生时要调用的函数。每个回调函数都接受如下两个参数：

reactor_object 调用回调函数的 VLR 对象。

list 与特定事件相关联的其他数据元素。该表针对特定事件的内容见表 “LISP 反应器回调数据”。

返回值

reactor_object 参数。

LISP 反应器事件

事件名称

说明

:vlr-lispWillStart

即将对 AutoLISP 表达式进行求值。

:vlr-lispEnded

已完成对 AutoLISP 表达式的求值。

:vlr-lispCancelled

已取消对 AutoLISP 表达式的求值。

LISP 反应器回调数据

名称

表长度

参数

:vlr-lispEnded

:vlr-lispCancelled

0

:vlr-lispWillStart

1

字符串，包含要求值的 AutoLISP 表达式的第一行。
