明经通道 外部定义命令  
saveimg
 

将渲染图形保存到 BMP、TGA 或 TIFF 格式的文件中（外部定义：render ARX 应用程序）

(c:saveimg filename type [portion] [xoff yoff xsize ysize] [compression])

当 AutoCAD 被配置为渲染独立视口时，portion 参数不予使用。可以指定图像的大小和偏移。对于 TGA 和 TIFF 文件，还可以指定压缩方式。

参数

saveimg 的参数说明如下表：

SAVEIMG 函数参数

参数

数据类型

说明

缺省值

STR

无

STR

无

STR

要保存的屏幕区域：

A - 活动视口

D - 绘图区域

F - 全屏

注意 此参数现在被忽略，但由于脚本兼容性的缘故仍向用户提供。

“A”

xoff

INT

X 偏移（以像素点为单位） 

0

yoff

INT

Y 偏移（以像素点为单位） 

0

INT

X 尺寸（以像素点为单位） 

实际 X 尺寸

ysize

INT

实际 Y 尺寸

compression

STR

压缩设置：

NONE（不压缩）

PACK（仅适用于 TIFF 文件）

RLE（仅适用于 TGA 文件）

无

示例

下例以文件 test.tif 保存全屏 TIFF 图像，该文件不压缩：

(c:saveimg "TEST" "TIF" "NONE")
