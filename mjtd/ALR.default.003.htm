明经通道 AutoLISP 函数  
+ (加)
 

返回所有数的和

(+ [number number] ...)

参数

number

数值型。

返回值

所有数相加的结果。如果在调用本函数时仅提供了一个 number 参数，本函数会返回它和 0 相加的结果，即返回该 number。如不提供参数，则返回 0。

示例

(+ 1 2)                   返回  3 
(+ 1 2 3 4.5)             返回  10.5
(+ 1 2 3 4.0)             返回  10.0
