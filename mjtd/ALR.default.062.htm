arguments

AutoCAD 命令、选项或表达式?/p>

command 函数的参数可以是字符串、实数、整数或点，但必须与要执行的命令所需的参数一致。空字符?("") 表示从键盘键?ENTER 键。不带参数调?command 相当于键?ESC 键，这样可取消大多数 AutoCAD 命令?/p>

command 函数将每一个参数顺序传?AutoCAD 以响应提示。它以字符串形式提交命令名和选项；以两个实数组成的表的形式提交二维点；以三个实数组成的表的形式提交三维点。只有在命令提示?AutoCAD 才能识别命令名?/p>

要注意的是，如果?Visual LISP 中使?command 函数，控制并不会转移?AutoCAD 中。如果命令要求用户输入，则可以在控制台窗口中看到返回结果 (nil)，但 AutoCAD 将等待输入，必须手动激?AutoCAD 并响应提示。在这样做以前，任何后续命令都会失败?/p>

nil

限制和注意事?/p>

AutoCAD ?SKETCH 命令直接读取键盘和数字化仪，因此不能?AutoLISP ?command 函数中使用这两个命令。如果在函数中使?SCRIPT 命令，它必须是出现在?AutoLISP 程序的最后?/p>

同样，如果在 acad.lsp ?MNL 文件中使?command 函数，则只能?defun 表达式中调用。应使用 S::STARTUP 函数定义在绘图任务开始时就要执行的那些初始化命令?

对于那些需要选取一个对象的 AutoCAD 命令（如 BREAK ?TRIM 命令），可以提供一个用 entsel 函数获得的表来取代交互操作时为选取一个对象而需要的点，这方面的例子可参? 《Visual LISP 开发人员手册》中?将拾取点传给 AutoCAD 命令 ?/p>
