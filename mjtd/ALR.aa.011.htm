明经通道 外部定义命令  
matlib
 

管理材质库（外部定义：render ARX 应用程序）

(c:matlib mode name [file])

参数

mode

字符串，指定此函数执行的操作。有效值为：

I 从材质库中输入材质。

E 向材质库中输出材质。

D 从图形中删除材质。

C 从图形中删除未附着的材质。

L 列出材质。

name

字符串，指定要输入、输出或删除的材质的名称。

file

字符串，指定材质库文件的名称。file 参数必须包括 .mli 文件扩展名。

示例

从标准 AutoCAD 渲染材质库 render.mli 中输入材质 BRASS：

命令： (c:matlib "I" "brass" "c:/autocad/support/render.mli")
1

“删除”模式中不使用 file 参数。

(c:matlib "D" "steel")
