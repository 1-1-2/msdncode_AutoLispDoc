明经通道 Visual LISP 函数  
vl-bb-set
 

设置黑板名称空间中的变量

(vl-bb-set ’symbol value)

参数

’symbol

符号，命名要设置的变量。

value

除函数外的任意值。

返回值

指定给 symbol 的 value。

示例

命令： (vl-bb-set 'foobar "Root toot toot")

"Root toot toot"

命令： (vl-bb-ref 'foobar)

"Root toot toot"

参阅

vl-bb-ref 函数。关于黑板变量空间的说明，请参见 《Visual LISP 开发人员手册》中的在名称空间中共享数据。
