外部定义命令
 

由 ObjectARX® 或 AutoLISP® 应用程序定义的 AutoCAD® 命令称为外部定义命令。AutoLISP 应用程序访问外部定义命令的方式与访问内置 明经通道 AutoLISP 函数的方式不同。许多外部定义命令都具有自己的编程接口，通过这些接口 AutoLISP 应用程序可以充分利用它们的功能。
关于本附录中所描述命令的详细信息，请参见《命令参考》。

本章内容包括：

  • 3dsin  输入 3D Studio (.3ds) 文件（外部定义：render ARX 应用程序）
  • 3dsout  输出 3D Studio 文件（外部定义：render ARX 应用程序）
  • align  平移和旋转对象，使它们与其他对象对齐（外部定义：geom3d ARX 应用程序）
  • cal  调用联机的几何图形计算器并返回表达式的值（外部定义：geomcal ARX 应用程序）
  • fog  加大对象与相机之间的距离（外部定义：render ARX 应用程序） 
  • light  创建、修改和删除光源和照明效果（外部定义：render ARX 应用程序）
  • lsedit  创建或修改配景对象（外部定义：render ARX 应用程序）
  • lslib  管理配景库（外部定义：render ARX 应用程序）
  • lsnew  创建配景对象（外部定义：render ARX 应用程序）
  • matlib  管理材质库（外部定义：render ARX 应用程序）
  • mirror3d  相对于用户指定的平面反射选定的对象（外部定义：geom3d ARX 应用程序）
  • render  用几何图形、光源和表面修饰信息创建三维线框模型的真实感着色图像（外部定义：render ARX 应用程序）
  • renderupdate  在下一次渲染时生成 en2face 文件（外部定义：render ARX 应用程序）
  • replay  显示 BMP、TGA 或 TIFF 图像（外部定义：render ARX 应用程序）
  • rmat  创建、编辑、附着和拆离渲染材质（外部定义：render ARX 应用程序）
  • rotate3d  绕任意三维轴旋转对象（外部定义：geom3d ARX 应用程序）
  • rpref  设置渲染系统配置（外部定义：render ARX 应用程序）
  • saveimg  将渲染图形保存到 BMP、TGA 或 TIFF 格式的文件中（外部定义：render ARX 应用程序）
  • scene  （仅适用于图纸空间）创建新场景，修改或删除现有的场景（外部定义：render ARX 应用程序）
  • setuv  向选定对象指定材质贴图坐标，此函数有两种模式，由字符串参数指定（外部定义：render ARX 应用程序）
  • showmat  列出选定对象的材质类型和附着方法（外部定义：render ARX 应用程序）
  • solprof  创建三维实体的剖面图像（外部定义：solids ARX 应用程序）
  • stats  显示上一次渲染的统计信息（外部定义：render ARX 应用程序）
