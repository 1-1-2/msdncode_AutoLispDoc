明经通道 外部定义命令  
rpref
 

设置渲染系统配置（外部定义：render ARX 应用程序）

(c:rpref mode option [setting])

c:rpref 函数决定渲染所使用的参数和缺省的渲染操作。

参数

mode

字符串，有效值为：

DEST 目标为：视口、渲染窗口或文件

ICON 光源和材质图标块的缩放比例

ROPT 其他渲染选项

SELECT 是否提示选择对象

STYPE 渲染类型：一般渲染、照片级真实感渲染或照片级光线跟踪渲染

TOGGLE 渲染选项

option

取决于 mode。

setting

取决于 mode。

参阅 设置“渲染到文件”选项。

DEST—目标配置

选择所使用的输出设备。

(c:rpref "DEST" option)

参数

option

字符串，指定渲染目标。有效值为：

FRAMEBUFFER 渲染到视口

HARDCOPY 渲染到渲染窗口

FILE 渲染到文件

示例

如下调用指定渲染到文件：

(c:rpref "DEST" "FILE")

ICON—图标配置

指定图形中光源或材质图标块的大小。

(c:rpref "ICON" option)

参数

option

实数，指定图标块的大小。缺省值为 1.00。

示例

如下函数调用将图标比例改为 50%：

(c:rpref "ICON" 0.5)

STYPE—渲染类型配置

指定所使用的渲染类型。

(c:rpref "STYPE" option)

参数

option

字符串，指定渲染类型。有效值为：

ARENDER 一般渲染

ASCAN 照片级真实感渲染

ARAY 照片级光线跟踪渲染

示例

如下代码指定下一次渲染类型为一般 AutoCAD 渲染：

(c:rpref "STYPE" "ARENDER")

SELECT—选择配置

指定是否在渲染前提示选择对象。

(c:rpref "SELECT" option)

参数

option

字符串，指定是否提示。有效值为：

ALL 渲染整个场景

ASK 提示选择对象

示例

如下调用将渲染配置为提示用户选择对象：

(c:rpref "SELECT" "ASK")

TOGGLE—开关配置

控制不同的渲染选项。

(c:rpref "TOGGLE" option setting)

参数

option

字符串，指定是否提示。有效值为：

CACHE 渲染到缓存文件。只要图形中的几何图形或视图未改变，缓存文件就可以用于随后的渲染，这可以节省重新镶嵌时所需的时间。

SHADOW 带阴影渲染。

SMOOTH 平滑着色渲染。

MERGE 将对象与背景合并。

FINISH 应用材质。

SKIPRDLG 不显示“渲染”对话框。

setting

字符串，指定开关的状态。setting 的有效值为“ON”和“OFF”。 

示例

如下调用关闭合并选项并打开阴影选项：

(c:rpref "TOGGLE" "MERGE" "OFF")
(c:rpref "TOGGLE" "SMOOTH" "ON")
