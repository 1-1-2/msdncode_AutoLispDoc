明经通道 AutoLISP 函数  
write-char
 

将一个字符写到屏幕上或一个已打开的文件中

(write-char num [file-desc])

参数

num

要输出字符的十进制 ASCII 码。

file-desc

已打开文件的文件描述符。

返回值

num 参数。

示例

下例将字符 C 写到命令窗口，并返回提供的 num 参数：

命令： (write-char 67)

C67

假定 f 是某打开文件的描述符，则下面的命令将字符 C 写入该文件：

命令： (write-char 67 f)

67

请注意 write-char 不能将 NULL 字符（ASCII 代码为 0）写入文件。

参阅

关于 ASCII 码列表，请参见《自定义手册》。
