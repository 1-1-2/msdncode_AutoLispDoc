明经通道 外部定义命令  
replay
 

显示 BMP、TGA 或 TIFF 图像（外部定义：render ARX 应用程序）

(c:replay filename type [xoff yoff xsize ysize])

通过 REPLAY 命令，可在 AutoCAD 渲染时显示 BMP、TGA 或 TIFF 文件。用此命令的功能使图像在多种偏移和尺寸下显示。

参数

filename

字符串，指定图像文件名。

type

字符串，指定文件类型。有效值为 BMP、TGA 或 TIFF。

xoff

整数，指定图像的 X 偏移（以像素点为单位）。缺省为 0。

yoff

整数，指定图像的 Y 偏移（以像素点为单位）。缺省为 0。

xsize

图像的 X 尺寸（以像素点为单位）。缺省为实际 X 尺寸。

ysize

图像的 Y 尺寸（以像素点为单位）。缺省为实际 Y 尺寸。

示例

如下调用显示名为 test.tga 的图像，图像从左下角开始显示，具有零偏移，宽度为 500 像素，高度为 400 像素：

(c:replay "TEST" "TGA" 0 0 500 400)
