明经通道 AutoLISP 函数  
arxload
 

加载 ObjectARX 应用程序

(arxload application [onfailure])

参数

application

用双引号引起来的字符串，或包含可执行文件名的变量。可以省略文件名中的后缀 .arx。

必须提供 ObjectARX 可执行文件的全路径名，除非该文件是在 AutoCAD 支持文件搜索路径中。

onfailure

加载失败时要执行的表达式。

返回值

如果成功则返回应用程序名。如果不成功，若提供了 onfailure 参数，函数返回该参数的值，否则返回相应的错误信息。

如果试图加载一个已经加载的 ObjectARX 应用程序，arxload 将发出一条错误信息。在使用 arxload 加载一个应用程序之前，可以调用 
arx 函数检查已加载的 ARX 应用程序。

示例

加载 AutoCAD 安装路径下的文件 acbrowse.arx：

命令: (arxload "c:/program files/ <AutoCAD installation directory>/acbrowse.arx")

"c:/program files/ <AutoCAD installation directory>/acbrowse.arx"

参阅

arxunload 函数。
