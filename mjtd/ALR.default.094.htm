明经通道 AutoLISP 函数  
eq
 

确定两个表达式是否具有相同的约束条件

(eq expr1 expr2)

eq 函数确定表达式 expr1 和 expr2 是否设置为同一对象（例如，用 setq 函数设置）。

参数

expr1

用于比较的表达式。

expr2

与 expr1 进行比较的表达式。

返回值

如果表达式相同则返回 T，否则返回 nil。

示例

给定如下指令：

(setq f1 '(a b c)) 
(setq f2 '(a b c)) 
(setq f3 f2)

比较 f1 和 f3：

命令： (eq f1 f3)

nil

因为 f1 和 f3 虽然值相同，但不指向同一个表，所以 eq 返回 nil。

比较 f3 和 f2：

命令： (eq f3 f2)

T

因为 f3 和 f2 指向同一个表，所以 eq 返回 T。

参阅

  = (等于) 和 equal 函数。
