明经通道 Visual LISP 函数  
vl-vbarun
 

运行 Visual Basic 宏

参数

(vl-vbarun  macroname)

macroname

字符串，表示加载的 Visual Basic 宏。

返回值

macroname

示例

加载 VBA 工程文件：

_$ (vl-vbaload "c:/program files/<AutoCAD 安装目录>/sample/vba/drawline.dvb")
"c:\\program files\\<AutoCAD 安装目录>\\sample\\vba\\drawline.dvb"

运行加载工程中的宏：

_$ (vl-vbarun "drawline")
"drawline"

参阅

vl-vbaload 函数。
