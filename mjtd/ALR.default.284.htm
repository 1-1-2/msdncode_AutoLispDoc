明经通道 Visual LISP 函数  
vl-load-com
 

将 Visual LISP 扩展功能加载到 AutoLISP

(vl-load-com)

该函数加载 Visual LISP 提供的扩展 明经通道 Visual LISP 函数。Visual LISP 扩展通过 AutoLISP 实现对 ActiveX 和 AutoCAD 反应器的支持，同时还提供了 ActiveX 实用程序、数据转换函数、词典处理函数和曲线测量函数。

如果扩展已经加载，vl-load-com 不做任何工作。

返回值

未确定。

load 函数以及《Visual LISP 开发人员手册》中的 使用扩展 明经通道 Visual LISP 函数。
