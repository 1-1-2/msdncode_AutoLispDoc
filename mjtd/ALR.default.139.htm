明经通道 AutoLISP 函数  
initdia
 

强制显示下一个命令的对话框

(initdia [dialogflag])

目前，下列命令可以使用 initdia 函数：ATTDEF、ATTEXT、BHATCH、BLOCK、COLOR、IMAGE、IMAGEADJUST、INSERT、LAYER、LINETYPE、MTEXT、PLOT、RENAME、STYLE、TOOLBAR 和 VIEW。

参数

dialogflag

整数型。如果没有提供该参数或提供的该参数不为 0，下一个（也仅有下一个）命令将使用该命令的对话框而不是命令行提示。

如果 dialogflag 参数为 0，那么该调用将清除以前对该函数的调用，恢复显示命令行界面的缺省状态。

返回值

nil

示例

首先，在不调用 initdia 的情况下发出 PLOT 命令：

命令： (command "_.PLOT")

plot

输入布局名 <Model>: nil

输入布局名 <Model>:

AutoCAD 提示用户在命令窗口进行输入。

使用下列函数调用显示“打印”对话框：

(initdia)
(command "_.PLOT")
