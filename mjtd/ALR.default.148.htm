明经通道 AutoLISP 函数  
list
 

将任意数目的表达式组合成一个表

(list [expr...])

本函数常用于定义二维或三维点变量（由两个或三个实数组成的表）。

参数

expr

AutoLISP 表达式。

返回值

表。如果未提供表达式，则 list 返回 nil。

示例

_$ (list 'a 'b 'c)
(A B C) 
_$ (list 'a '(b c) 'd)
(A (B C) D) 
_$ (list 3.9 6.7)
(3.9 6.7)

如果表中没有变量和未定义的项，还可以用 quote 函数显式引用一个表，这和调用 list 函数效果相同。单引号 ( ' ) 被定义为 quote 函数。

_$ '(3.9 6.7) 等于  (list 3.9 6.7)

这对创建关联表和定义点来说很有用。

参阅

quote、vl-list* 和 vl-list-length 函数。
