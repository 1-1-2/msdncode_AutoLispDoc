明经通道 AutoLISP 函数  
C 函数
 

本节内容包括:

  • caddr  返回表的第三个元素
  • cadr  返回表的第二个元素
  • car  返回表的第一个元素
  • cdr  返回去掉了第一个元素的表
  • chr  将代表字符 ASCII 码的整数转换成包含相应的单一字符的字符串
  • client_data_tile  将特定应用数据与一个对话框控件相关联
  • close  关闭一个已打开的文件
  • command  执行一条 AutoCAD 命令
  • cond  多条件、多处理结果函数
  • cons  向表的头部添加一个元素，或构造一个点对
  • cos  返回弧度角度的余弦值
  • cvunit  在不同测量单位之间进行换算
