外部定义命令
 
明经通道 外部定义命令  
3dsin
 

输入 3D Studio (.3ds) 文件（外部定义：render ARX 应用程序） 

(c:3dsin mode [multimat create] file)

参数

mode

整数，用于指定命令的使用方式：交互式 (mode = 1) 或非交互式 (mode = 0)。

multimat

整数，指定如何处理具有多种材质的对象。仅当 mode 设置为 0 时需要。有效值为：

0 为每种材质创建一个新对象

1 将第一种材质分配给新对象

create

整数，指定如何组织新对象。此模式总是输入 .3ds 文件中的所有对象。仅当 mode 设置为 0 时需要。有效值为：

0 为每个 3DS 对象创建一个图层

1 为每种 3DS 颜色创建一个图层

2 为每种 3DS 材质创建一个图层

3 将所有新对象都放在一个图层上 

file

字符串，指定要输入的 .3ds 文件。必须包括 .3ds 文件扩展名。

模式 0 总是输入 .3ds 文件中的所有对象。

示例

打开 3D Studio 文件 globe.3ds 用于输入，并提示用户输入时应注意的细节：

(c:3dsin 1 "globe.3ds")

输入 shadow.3ds 文件，三个参数值分别表示：禁止用户输入（非交互式），拆分具有多种材质的对象，并将所有新对象放在同一个图层上。

命令： (c:3dsin 0 0 3 "c:/my documents/cad drawings/shadow.3ds")
