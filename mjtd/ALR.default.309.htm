明经通道 Visual LISP 函数  
vl-string-search
 

在字符串中搜索指定子串

(vl-string-search pattern string [start-pos])

参数

pattern

字符串，包含要搜索的子串。

string

字符串，在其中搜索 pattern。

start-pos

整数型，表示搜索的开始位置。缺省值为 0。

返回值

整数，表示找到指定 pattern 的位置。如果未找到子串，则返回 nil。字符串第一个字符的位置为 0。

示例

_$ (vl-string-search "foo" "pfooyey on you")
1
_$ (vl-string-search "who" "pfooyey on you")
nil
_$ (vl-string-search "foo" "fooey-more-fooey" 1)
11
