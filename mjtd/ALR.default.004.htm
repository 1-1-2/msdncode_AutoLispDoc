明经通道 AutoLISP 函数  
– (减)
 

将第一个数减去其他数的和并返回差值

(C [number number] ...)

参数

number

数值型。

返回值

相减所得的结果。如果提供的 number 参数多于两个，则本函数返回从第一个数减去第二个数到最后一个数的和的差。如果仅提供一个 number 参数，本函数返回 0 减去它的差，即返回 -number。如不提供参数，则返回 0。

示例

(- 50 40)                 返回  10
(- 50 40.0)               返回  10.0
(- 50 40.0 2.5)           返回  7.5
(- 8)                     返回  -8
