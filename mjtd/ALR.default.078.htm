dimy_tile
 

返回控件的高度

(dimy_tile key)

参数

key

字符串，指定要查询的控件的操作名。key 参数区分大小写。

返回值

控件高度。

返回的坐标值是指定控件所允许的最大值。由于控件的坐标值是从 0 开始计算的，所以，假设 Y 是总高度，则函数返回的坐标值不会大于 Y-1。dimx_tile 和 dimy_tile 
函数一般与 vector_image、fill_image 和 slide_image 函数配套使用，它们可为这些函数提供所需的指定控件大小的绝对坐标。

示例

(setq tile_height (dimy_tile "my_tile"))
