外部定义命令
 
明经通道 外部定义命令  
3dsout
 

输出 3D Studio 文件（外部定义：render ARX 应用程序） 

(c:3dsout sset omode div smooth weld file)

参数

sset

选择集，包含要输出的 AutoCAD 对象。

omode

整数（0 或 1），指定 AutoCAD 数据表示法的输出模式。当前 omode 设置为 0 或 1 时，3dsout 输出结果都相同。

div

整数，指定如何将 AutoCAD 对象拆分为 3D Studio 对象。有效值为：

0 为每个 AutoCAD 图层创建一个对象

1 为每种 AutoCAD 颜色创建一个对象

2 为每种 AutoCAD 对象类型创建一个对象

smooth

整数，指定自动平滑的阈值角。如果 smooth 设置为 -1，则不进行自动平滑。如果设置为 0-360，则当平面法线之间的角度小于此值时，AutoCAD 产生平滑效果。

weld

实数，指定相邻顶点接合时的距离阈值。如果 weld 设置为小于 0 的值，则禁止接合。如果设置为大于或等于 0 的值，则 AutoCAD 接合间距小于此值的顶点。

file

字符串，指定要创建的 3D Studio 文件名。必须包括 .3ds 文件扩展名。

示例

输出所有图形，按图形的图层创建 3D Studio 对象，并使用 30 度的平滑阈值和 0.1 的接合距离：

(c:3dsout (ssget "X") 0 0 30 0.1 "testav.3ds")
