明经通道 AutoLISP 函数  
regapp
 

为当前 AutoCAD 图形注册一个应用名，为使用扩展对象数据作准备

(regapp application)

参数

application

命名应用程序的字符串。名称必须为有效的符号表名称。关于 AutoLISP 确定符号名有效性的规则，请参见 snvalid 的说明。

返回值

如果已经注册了一个同名的应用名，该函数返回 nil，否则它返回应用名。

如果注册成功，该应用名会被加入到 APPID 符号表中，该表保留了该图形中所有使用扩展数据的应用名。

Examples 

(regapp "ADESK_4153322344")
(regapp "DESIGNER-v2.1-124753")

注意 建议您使用唯一的应用名。一个可以保证其唯一性的命令方法是使用公司和产品名以及一个唯一的数值串（如您的电话号码或当前日期/时间）。也可以将产品的版本号加入到应用名中或存储在应用程序定义数据的一个单独的整数或实数域中，如 (1040 2.1)。
