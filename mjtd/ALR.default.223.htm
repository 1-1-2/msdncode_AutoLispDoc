明经通道 AutoLISP 函数  
start_image
 

开始对对话框控件中的一个图像控件进行操作

(start_image key)

调用本函数之后，可以调用 fill_image、slide_image 和 vector_image 等函数对图像控件进行各种操作，直到应用程序调用 end_image 函数才结束。

参数

key

字符串，指定对话框控件。key 参数区分大小写。

返回值

如果成功则返回 key 参数，否则返回 nil。

注意 不要在 start_image 和 end_image 函数的调用之间使用 set_tile 函数。
