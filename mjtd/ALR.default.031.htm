明经通道 AutoLISP 函数  
alert
 

显示一个警告框，其中显示一条出错或警告信息

(alert string)

参数

string

显示在警告框中的字符串。

返回值

nil

示例

下列代码在警告框中显示一条信息：

(alert "该函数不可用。")

通过在 string 参数中使用换行符可以在一个警告框中显示多行字符：

(alert "该函数\n不可用。")

注意 警告框中所能显示的信息的行数以及每行的长度依赖于所使用的平台、设备和窗口。 AutoCAD 将截去超长的字符以适应警告框的大小。
