明经通道 外部定义命令  
lsedit
 

创建或修改配景对象（外部定义：render ARX 应用程序）

(c:lsedit mode [options])

这种形式的 c:lsedit 用于创建或修改图形中的配景对象实例。

(c:lsedit "LIST" object)

这种形式的 c:lsedit 列出指定配景对象的属性。返回的列表包括指定对象的名称、高度、位置和对齐方式。

(c:lsedit object height [position [alignment]])

这种形式的 lsedit 修改配景对象。

参数

LSEDIT 命令的参数说明如下表：

LSEDIT 参数

参数

数据类型

说明

缺省值

object

ENAME

配景对象的句柄 

无

height

REAL

对象的高度（以图形单位表示）。如果为空，则当前值不改变。

无

position

LIST (of reals)

对象基面（点）的位置。如果为空，则当前值不改变。

无

alignment

INT

指定项目的几何图形和对齐方式：

0 - 单一表面相机对齐

1 - 单一表面非相机对齐

2 - 跨越表面非相机对齐

3 - 跨越表面相机对齐

如果为 nil，则当前值不改变。

无

示例

修改配景对象，其中 <ename> 为要修改的对象的 AutoCAD 名称 (entsel)，对齐方式保持不变。

(c:lsedit <ename> 35.0 ’(10.0 23.0) nil)
