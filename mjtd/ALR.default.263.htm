明经通道 Visual LISP 函数  
vl-doc-set
 

设置当前文档名称空间中变量的值

(vl-doc-set ’symbol value)

该函数可以由 VLX 应用程序使用，用于设置当前文档名称空间中变量的值。

在文档名称空间中执行 vl-doc-set 等效于 set。

参数

’symbol

指定变量的符号。

value

任意值。

返回值

设置的 value。

示例

命令： (vl-doc-set 'foobar "Rinky dinky stinky")

"Rinky dinky stinky"

参阅

  vl-doc-ref 函数。
