明经通道 Visual LISP 函数  
vlax-create-object
 

创建应用程序对象的新实例

(vlax-create-object prog-id)

在需要开始应用程序的新实例时，可以使用 vlax-create-object 创建由 <Component> 指定类型的对象（请参见参数说明）。要使用当前的实例，可以使用 vlax-get-object。但是，如果应用程序对象已经将自己注册为单一实例对象，则无论调用多少次 vlax-create-object，都只能创建该对象的一个实例。

参数

prog-id

字符串，包含需要的 ActiveX 对象的可编程标识符。prog-id 的格式为：

<Vendor>.<Component>.<Version>

For example:

AutoCAD.Drawing.15

返回值

应用程序对象（VLA 对象）。

示例

创建 Excel 应用程序的实例：

_$  (vlax-create-object "Excel.Application")
#<VLA-OBJECT _Application 0017b894>
