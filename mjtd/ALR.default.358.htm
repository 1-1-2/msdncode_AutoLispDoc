明经通道 Visual LISP 函数  
vlax-map-collection
 

将函数应用到集合中的所有对象

(vlax-map-collection objfunction)

参数

obj

代表集合的 VLA 对象。

function

要应用到 obj 的符号或 lambda 表达式。

返回值

第一个参数 obj。

示例

(vlax-map-collection (vla-get-ModelSpace acadDocument) 'vlax-dump-object)
