明经通道 Visual LISP 函数  
vlax-safearray-get-l-bound
 

返回 Safearray 某一维的下界（开始索引）

(vlax-safearray-get-l-bound var dim)

参数

var

变量，其数据类型为 SafeArray。

dim

数组的维数。第一维为 1。

返回值

整数，表示维的下界（开始索引）。如果 var 不是数组或 dim 无效（如设置为 0 或大于数组维数的数值），则返回错误信息。

示例

下例对如下定义的 SafeArray 求值：

(vlax-make-safearray vlax-vbString '(1 . 2) '(0 . 1) ))

获取数组第一维的开始索引值：

_$ (vlax-safearray-get-l-bound tmatrix 1)
1

第一维以索引 1 开始。

获取数组第二维的开始索引值：

_$ (vlax-safearray-get-l-bound tmatrix 2)
0

第二维以索引 0 开始。

参阅

vlax-make-safearray、vlax-safearray-get-dim 和 vlax-safearray-get-u-bound 函数。
